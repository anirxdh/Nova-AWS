# Milestones

## v1.0 — Voice-First Q&A Assistant (Global Engineering Hackathon 2026)

**Shipped:** March 2026

**What was built:**
- Push-to-talk Chrome extension (hold backtick, speak, release)
- Real-time waveform indicator near cursor
- Groq Whisper STT + Llama 4 Vision for screen-aware Q&A
- Streaming text overlay with Shadow DOM isolation
- ElevenLabs TTS + Web Speech API fallback
- Conversation memory (20 turns per tab)
- 5 explanation levels (Kid → Executive)
- 3 display modes (Both, Audio Only, Text Only)
- Settings page (React), popup, welcome page

**Phases:** 1–5 (retroactive; built during hackathon without GSD)

**Outcome:** Working demo; Q&A only, no task execution
